# 统计

## 总览

> https://api.dida365.com/api/v3/user/ranking

*请求的方式：GET*

成功响应：

```json
{
  "ranking": 1.72,
  "taskCount": 22,
  "projectCount": 3,
  "dayCount": 292,
  "completedCount": 5,
  "score": 20,
  "level": 1
}
```

响应字段说明：
|字段|值|作用|
|----|---|----|
|ranking|float|你比`ranking`%的用户更勤奋|
|taskCount|int|任务数量|
|projectCount|int|项目数|
|dayCount|int|使用天数|
|completedCount|int|已完成任务数|
|score|int|我的成就值|
|level|int|账号等级|
